Justificativa das Escolhas


Servidor Dell T140 – servidor de dados e máquina virtual:
É indicado o servidor Dell T140 para servidor de dados e máquina virtual.
Ele possui recursos suficientes para manter os dados da empresa seguros e íntegros e rodar
uma máquina virtual com Linux para instalação dos softwares desenvolvidos pela empresa.
Sua configuração é um processador Intel Xeon E-2236 Six Core e 12 threads, memória de 16GB
DDR4 3200MT/s e 4 TB de HD SATA cabeado. Este servidor atende as necessidades de
segurança, integridade e recursos que a ZNet precisa.


Servidor Dell T140 – Servidor Firewall PfSense:
É indicado outro servidor Dell T140 para executar a função de firewall, este servidor é
praticamente idêntico ao servidor de dados, porém com recursos menores. Este servidor
possui 8GB de RAM DDR4 3200MT/s um processador Intel Xeon E-2224 quad-core e 4 threads
e um HD de 1TB. Escolhemos esse servidor, pois com ele podemos instar o PfSense e a partir
dele conseguimos administrar o consumo de banda larga para os computadores da rede,
monitorar os acessos dos funcionários, bloquear acessos estranhos entre outras funções que
um servidor PfSense oferece a empresa.


Nobreak TS Shara UPS Senno:
Como a empresa ZNet é uma empresa do ramo de serviços digitais, ela não poderá correr o
risco de ter seus serviços inativos caso aconteça de faltar energia no local onde a empresa será
instalada.
Tendo isso em vista, escolhemos um Nobreak de grande porte da TS Shara. Este Nobreak é um
Nobreak Senoidal de 3200VA, ele entrega 2240 Watts, este modelo vem instalado nele 2
baterias, porém ele tem espaço suficiente para acomodar 4 baterias veicular de 36 a 55 Ah,
podendo oferecer até 30 horas de energia.


Dell Vostro Small Desktop:
Como workstation, é indicado o computador Dell Vostro.
Este computador tem um processador Intel Core i5 de 10° geração hexa-core, 8GB DDR4 de
memória e um armazenamento SSD NVMe de 256GB. É um computador com bom
desempenho e recursos ideais para as atividades dos desenvolvedores da ZNet, seja
desenvolvimento de sistemas, design ou marketing. Ele possui um SSD de 256 GB podendo ser
pouco espaço para armazenamento das atividades dos colaboradores, porém tendo em vista o
servidor de dados para armazenamento de arquivos dos colaboradores, este SSD é o suficiente
para o uso. Caso não seja, o Dell Vostro tem espaço para a instalação de mais 2 HDs em seu interior.


Monitor Dell P2219H:
Em conjunto com o Dell Vostro, escolhemos o monitor Dell P2219h.
Ele é um monitor LCD de 21.5” com resolução nativa de 1920 x 1080 a 60hz e o tipo de painel é
o IPS e tela antiofuscante.
Este monitor é um monitor ideal para trabalho, principalmente pelo painel ser IPS, com isso
ele acaba tendo uma qualidade de imagens e cores bem melhor do que o painel TN e também
independente do ângulo que a pessoa olhar para a tela ele não ofusca e nem muda de cor, ele
possui uma frequência de trabalho de 60hz e um tempo de resposta de 8ms normal e 5ms rápido.


Teclado e mouse sem fio Dell Pro:
É um kit básico sem fio, porém suficientes para as atividades dos colaboradores de forma que
atenda a necessidade e conforto deles.


Rack servidor organizador:
Para equipamentos de rede e servidor é indicado uma rack de servidor de grande porte
podendo comportar os 2 servidores e o Nobreak, e nas prateleiras superiores os equipamentos
de rede e com espaço para instalação de patch painel


Patch Painel:
Para conexão de rede é indicado um patch painel de 24 portas compatível com plugs RJ45 e
RJ11 ideal para interligar os equipamentos de rede com os demais do local.


Switch gerenciável:
É indicado um switch gerenciável de 28 portas com 24 portas PoE+JetStream, este switch é um
switch Gigabit, ou seja, velocidade de banda ultrapassa os 100Mbps e possui roteamento
estático que ajuda a rotear o tráfego interno da internet.


Load Balance:
Para garantia de funcionamento seguro mesmo com problemas de internet, indicamos um
Load Balance TP-Link Gigabit. Esse equipamento é responsável por subir uma segunda internet
caso a primeira possua problemas e quedas, ele é um equipamento Gigabit com 1 porta LAN
Gigabit fixa 1 porta WAN Gigabit fixa e 3 portas Gigabit alternáveis LAN/WAN ideal para
segurança de rede da ZNet.
