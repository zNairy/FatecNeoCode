Sistema Operacional



O Windows Server 2012 R2 Standard é a versão Standard é indicada para ambientes
não virtualizados ou com baixa densidade de VMs.

Aqui, valem as mesmas especificações da versão Datacenter – a diferença está no número de VMs. Com a Standard, são permitidos 2 VMs por licença.

É interessante apontar, entretanto, que esse limite é relativo a licenciamento, e não a softwares. Para implementar mais de 2 VMs, será necessário adquirir licenças extras.

Esse sistema operacional será utilizado como principal, apenas nos servidores (será abordado posteriormente).


O Ubuntu Server 20.04 LTS O Ubuntu Server é um sistema operacional open source específico
para servidores que integra a gama de distribuições do Ubuntu
desenvolvida pela Canonical, como Ubuntu Desktop, Ubuntu Cloud e Ubuntu Core.

De acordo com pesquisa realizada pela W3Techs (ref. agosto de 2018),
14,2% dos websites no mundo utilizam o Ubuntu.

A popularidade do Ubuntu Server em data centers tem explicação, a começar pela qualidade do sistema operacional em si.

Esse sistema operacional será virtualizado no Windows Server 2012 R2 Standard, para instalação do sistema da zNet.


O Windows 10 Home muda o cenário da segurança, protegendo as identidades,
os dispositivos e as informações dos usuários com uma solução abrangente baseada
em inteligência que somente a Microsoft oferece.

Recursos integrados de segurança, produtividade e gerenciamento
economizam tempo, dinheiro e reduzem os problemas.

Esse sistema operacional já vem integrado na compra dos Desktops ou Workstations (será abordado posteriormente)



Requisitos Necessários

Requisitos para instalação do:

Windows Server 2012 Standard

— Processador: Mínimo de 1,4 GHz e 64 bits
— Memória RAM: Mínimo de 512 MB
— Espaço em disco mínimo: 32 GB
— Adaptador: Gigabit (10/100/1000baseT) Ethernet
— Monitor: Super VGA (1024 x 768) ou com resolução superior

Windows 10 Home

— Processador: 1 gigahertz (GHz) ou um processador mais rápido, ou SoC
— Memória RAM: 1 GB para 32 bits ou de 2 GB para 64 bits
— Espaço em disco mínimo: 16 GB para SO de 32 bits ou 20 GB para SO de 64 bits
— Placa Gráfica: DirectX 9 ou posterior com driver WDDM 1.0
— Vídeo: 800 x 600

Ubuntu Server 20.04 LTS

— Processador: 1 GHz CPU
— Memória RAM: 512 MB RAM (Memória do sistema)
— Espaço em disco mínimo: 2.5 GB


Justificativa

A virtualização é um processo imprescindivel para as companhias que buscam
por eficiencia e custos operacionais, nesse aspecto, a escolha do Windows
server traz atualizações e recursos de segurança e virtualização em tempo real.

A melhoria doMicrosoft Hype - V e também a repaginação do gerenciador
de tarefas do windows, é um desses exemplos.
Pois permite até mesmo que pequenas empresas consigam competir
com os data center de grandes companhias.

Alguns recursos também foram adicionados como proteção de conteúdo
e virtualização de hardware, melhorando gerenciamento da TI e performance na cloud.

Outros tipos de melhorias como backup em massa que evita que danos no servidor
comprometa dados também foram incrementados sendo o Windows server uma boa
opção que agrega custo e beneficio.

OBS: A estimativa para esse orçamento consiste em um periodo curto, considerando que apesar
do custo da versão 2012 ser atrativa, é aconselhável que servidores possuam um suporte de
correções de segurança por maior tempo, ou seja, tenha um tempo de vida de pelo menos 8 anos,
sendo portanto indicado a versão mais recente como a do windows server 2016.
